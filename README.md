# Plot_Leaflet
基于Leaflet(***ES6***)的标绘实现


## 基础环境
| 序号      | 名称      | 版本       |
| :------:  |:-------- | :----------|
|1          |Node      | ^v6.10.3LTS|
|2          |Typesctipt| ^2.3       |           
|3          |Webpack   | 2          |
|4          |Babel     | 2          |

## 获取
```
git clone https://github.com/Guzhongren/Plot_Leaflet.git
```

## 安装
1. 在 ***项目跟目录*** 执行如下shell
    ```
    npm install  #or yarn add 
    ```
2. 全局安装webpack和webpack-dev-server
    ```
    npm install -g webpack@2.6.1 webpack-dev-server@2.4.5
    ```
## 运行
运行
```
npm run dev
```
